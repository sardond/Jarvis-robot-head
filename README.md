# Jarvis-robot-head
This is code for my Jarvis,  (or whatever wake word you choose), robot head.
I wanted to have a robotic head that I could chat with, much like you could chat with Alexa. My inspiration came from finding Thomas Burns (alexatron) youtube video and Will Cogley's anamatronic eye mech. You will find build intructions on Intructables. I 3d printed the eye mech, but Will Cogley has a site with pre-built for sale. I built Version 1, but with a much different parts list, having me to modify/re-write the code. My current version 2 built has yet a different parts list and again re-write the code. This is the code I will post. Using a PI 10 inch screen helped alot while having PI terminal windows open and having a copilot, gemini, chatgpt, etc window open as I needed assistance getting the code right. I am by no means pro, but I enjoy learning and a challenge. Note: try to learn and pay attention to what the AI help is trying to do. Often times it was wrong, but mostly a huge help.

I will list from memory what hardware I used: PI5 8GB, SSD and hat, Uno R3, servo board, 6 sg90's, amp board and small speaker, 2 Yahboom power boards (12v to 5V), one deticated to the PI, Ugreen usb sound card, voltage isolator for the sound, 2 inch or 4 inch lcd, PI cam module 3 wide, 300mm cam cable, assorted wiring and M screw assortment box, assortment of 3.5mm audio cables, assortment of usb angle adapters. I purchased what I think will fit as I go over many weeks. The head is a robot mask on Amazon with lots of dremiling to fit everything.

3 seperate sketch's autorun when PI boots.
Sketch 1 is the Gemini brain. 
Sketch 2 is the face tracking.
Sketch 3 is the lcd sine-wave.

I will add these as I can.
